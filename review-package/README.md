# Review Package

This package contains a local copy of the website for review.

## Option 1: Run the project locally

Prerequisite:
- Install Node.js LTS

Steps:
1. Open a terminal in this folder.
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:4322`

## Option 2: View the built static version

If the `dist` folder is included in this package, you can run:

1. Open a terminal in this folder.
2. Run `npx serve dist`
3. Open the local URL shown in the terminal

## Included files

- `src/` site source files
- `scripts/` local helper script
- `package.json` and `package-lock.json`
- `astro.config.mjs`
- `tsconfig.json`
- `dist/` built output if available
